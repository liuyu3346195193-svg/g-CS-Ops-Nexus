# 客户运营中枢 (CS Ops Nexus) - StablePay

**专为 StablePay 首席客户官助理打造的数字化管理平台**

这是一个交互式的 Web 应用程序，旨在作为面试演示作品，展示候选人对 StablePay 业务痛点的理解以及如何通过数字化工具提升客户成功团队的运营效率。

## 核心功能

本系统围绕 CCO 助理的四大核心职能设计：

1.  **全局概览 (Dashboard)**
    *   **功能**: 实时监控关键业务指标（如商户入驻时长、活跃商户数、CSAT）。
    *   **价值**: 展现对业务健康度的敏感性，以及与公司 OKR（如“5分钟集成”）的对齐。

2.  **团队运营与标准化 (Team Ops)**
    *   **功能**: 内置 **AI SOP 生成器**，可一键生成标准化的流程文档（Markdown 格式），并支持项目进度追踪。
    *   **价值**: 体现流程管理能力和文档输出功底，解决团队知识传承问题。

3.  **商户入驻管道 (Onboarding Tracker)**
    *   **功能**: 类似看板的商户状态追踪，自动识别在 KYB 或集成阶段卡顿的商户。
    *   **价值**: 落实“梳理并优化商户注册路径”的职责，主动发现并解决问题。

4.  **智能知识库 (Knowledge Base)**
    *   **功能**: AI 驱动的工单回复助手，可根据上下文自动起草回复，并支持一键归档到 FAQ。
    *   **价值**: 提升客服响应效率，确保对外服务口径的一致性和专业性。

5.  **数据洞察 (Analytics Hub)**
    *   **功能**: 利用 AI 分析非结构化的客户反馈，提炼情感倾向和业务改进建议。
    *   **价值**: 为管理层提供基于数据的决策支持。

## 技术栈

*   **前端框架**: React 18 + TypeScript
*   **样式库**: Tailwind CSS
*   **图表组件**: Recharts
*   **AI 模型**: Google Gemini 2.5 Flash (用于生成 SOP、回复草稿及情感分析)
*   **图标库**: Lucide React

## 设计理念

*   **API 优先**: 界面设计强调开发者体验和集成效率。
*   **极简高效**: 采用清爽的 B 端 UI 风格，减少认知负荷。
*   **AI 赋能**: 在关键工作流中嵌入 AI 能力，而非为了 AI 而 AI。
